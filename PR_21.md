## Description

Add comprehensive AI utilities for multilingual support and semantic search with role-based features for SHO, Clerk, and Judge modules.

This PR introduces 12 advanced AI features including:
- Multilingual OCR with 11+ language support
- FAISS-based semantic search for case precedents
- BM25 keyword matching for section suggestions
- Cross-encoder reranking for relevance
- Enhanced legal NER for entity extraction
- Case readiness checking for SHO
- Document validation for Clerk
- Case brief generation for Judge
- Query expansion with legal synonyms
- Advanced document generation with templates
- Structured JSON logging
- Keyword suggestion from text
